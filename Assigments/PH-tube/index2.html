Var:

Var is a function that helps us to declare variables.It can be access through anywhere. Exaple - string,number,boolean etc.

Let:

Let is similar to var.But declaring a variable with let are block-scoped.

Const:

Const is also a way of declaring variable.But this time it can't be changed

Null:

Null is a assignment value. It can be assign to a variable as a representation of no value.

Undefined:

Undefine is the variable has been declare but it's value isn't assigned.
